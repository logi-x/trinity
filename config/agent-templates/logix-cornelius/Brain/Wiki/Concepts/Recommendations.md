# Recommendations

Surfacing **relevant courses, instructors, and content** to a learner —
personalization and discovery on the Experts platform.

## In the Experts app

- Built on semantic similarity over [[Wiki/Concepts/Embeddings]]
  (pgvector nearest-neighbour), optionally re-ranked with behavioural signals.
- Feeds curated home / catalog surfaces; part of the AI feature set —
  see [[Wiki/Concepts/AI Features]].
- Effectiveness is judged against engagement/conversion — see
  [[Wiki/Concepts/Analytics]].
