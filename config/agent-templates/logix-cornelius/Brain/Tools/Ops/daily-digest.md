# Operation - daily digest

Use when the user asks for recent external updates.

## Steps

1. Search the web for the requested topic and time window.
2. Save findings to `Inbox/Digest <YYYY-MM-DD>.md`.
3. Use one short item per source: title, source, date, one-line summary, likely wikilinks.
4. Do not ingest automatically.
5. Tell the user which items look worth ingesting.

## Rule

Digest is triage. Ingest is a separate operation.
