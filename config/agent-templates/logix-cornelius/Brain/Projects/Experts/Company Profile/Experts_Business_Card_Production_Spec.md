# Experts Business Card Production Spec

Illustrator-first production spec for Experts business cards

## Recommendation

Use `Illustrator` as the primary design tool.

It is the better fit for:

- a small-format print piece
- front/back card composition
- precise vector logo handling
- controlled bleed and print prep
- fast iteration on a single layout system

Use `InDesign` only if you later need:

- dozens of employee variants
- data merge
- a larger stationery system managed in one file

## Best workflow

1. design the master card in Illustrator
2. finalize front and back layouts
3. convert into named variants for each person
4. export print-ready PDF
5. if volume grows, consider moving the final system into InDesign for batch production

## Card size

For Saudi business cards, use one of these standard sizes:

- `90 x 50 mm`
- `85 x 55 mm`

Recommended for Experts:

- `90 x 50 mm`

Reason:

- slightly more elegant panoramic feel
- works better for bilingual information
- gives Arabic and English room to breathe

## Bleed and safe area

Set up the document as:

- final trim size: `90 x 50 mm`
- bleed: `3 mm` on all sides
- safe area: keep text at least `4 to 5 mm` inside trim

So your Illustrator artboard setup should effectively account for:

- artwork extending to `96 x 56 mm` including bleed

## Color mode

Work in:

- `CMYK`

Use RGB only if you are doing concept exploration first, then convert carefully before print.

## Brand direction

The card should feel:

- premium
- modern
- minimal
- bilingual
- confident without looking busy

Avoid:

- too many gradients
- overcrowded contact details
- tiny icons everywhere
- generic corporate blue layouts unrelated to Experts branding

## Brand colors

Use these as the base system:

- Experts primary accent: `oklch(0.58 0.17 255.97)` converted to CMYK equivalent by print proof
- logo leaf gradient: `#B1DF10` to `#5F8A02`
- logo blue gradient: `#2AAECF` to `#2A5E9B`
- logo wordmark gray: `#939393`

For print use, do this carefully:

- keep the official logo colors intact where possible
- use the purple-blue product accent as a secondary visual system, not as a replacement for the brand mark

## Typography

Recommended:

- English name/title: `DM Sans`
- English body/contact: `Inter`
- Arabic name/title: `Readex Pro`
- Arabic body/contact: `Almarai`

If printer/font packaging becomes a problem, outline only the final print file copy after approval.

## Layout recommendation

For Experts, the strongest approach is:

- `front`: logo-led, minimal, premium
- `back`: person details, bilingual

This keeps the card clean and avoids forcing too much text onto the brand side.

## Option A: Recommended layout

## Front

Use:

- centered or asymmetrically placed Experts logo
- lots of breathing room
- optional subtle purple-blue or dark background field

Possible front styles:

### Style 1

- white or near-white background
- Experts full logo centered
- very subtle abstract glow or fine grid texture

### Style 2

- dark charcoal background
- full logo centered
- subtle purple-blue edge glow

### Style 3

- minimal white background
- logo upper left
- short tagline lower right

Recommended:

- `Style 2` for a more premium modern feel

## Back

Use a clean bilingual contact structure.

Suggested information:

- full name in English
- full name in Arabic
- job title in English
- job title in Arabic
- phone
- email
- website
- optional QR code

## Information hierarchy

Prioritize this order:

1. name
2. title
3. phone
4. email
5. website

Do not overload with:

- full office address unless required
- too many social handles
- multiple phone numbers
- long slogans

## Bilingual arrangement

Best recommendation:

- English on left half
- Arabic on right half

Alternative:

- English stacked top
- Arabic stacked bottom

For Experts, because the company already works across Arabic and English, use:

- `left = English`
- `right = Arabic`

This matches the logic used in the company profile.

## Back layout structure

Suggested layout:

- top area: person name and title
- middle area: divider or spacing break
- bottom area: phone, email, website

Optional:

- QR code in one corner

## Grid

Use a simple structure:

- 2-column bilingual split on the back
- vertical alignment to a shared baseline
- one subtle divider optional, but not required

Keep spacing generous.

This card should feel calm, not packed.

## Logo handling

Use the official logo assets.

Prefer:

- vector logo if available
- if not, high-resolution transparent asset

Front:

- full logo preferred

Back:

- small icon or logo mark optional
- do not let it compete with the person’s information

## QR code

Optional but useful.

Best uses:

- website homepage
- personal Experts profile
- vCard/contact landing page

If you use a QR code:

- keep it small
- place it bottom right or bottom left
- do not let it dominate the back

## Printing notes

Ask the printer for:

- matte lamination front and back
- thick stock, ideally `350 gsm` or similar
- rounded corners only if the brand direction calls for it

Recommended finish:

- soft-touch or matte lamination

Optional premium upgrades:

- spot UV on logo
- foil on logo accent
- painted edges

Only use premium effects if they stay subtle. Experts should feel refined, not flashy.

## Illustrator setup

Create layers like this:

- `Guides`
- `Background`
- `Brand`
- `Text_EN`
- `Text_AR`
- `Icons_QR`
- `Notes`

Create two artboards:

- `Front`
- `Back`

If you plan multiple people, duplicate the back artboard and rename by person.

## Paragraph style logic

Even in Illustrator, keep text systemized.

Use these style categories:

- `Name_EN`
- `Title_EN`
- `Info_EN`
- `Name_AR`
- `Title_AR`
- `Info_AR`

## Suggested starting type sizes

These are starting points only:

- English name: `10.5 to 12 pt`
- English title: `7.5 to 8.5 pt`
- English contact: `7 to 8 pt`
- Arabic name: `11 to 13 pt`
- Arabic title: `8 to 9 pt`
- Arabic contact: `7.5 to 8.5 pt`

Arabic generally needs a little more optical room than English.

## Front concept recommendation

My recommendation for Experts:

### Front

- dark charcoal background
- centered Experts logo
- very faint purple-blue ambient glow
- no extra contact text

### Back

- light background
- bilingual contact layout
- English left, Arabic right
- small accent line or divider in Experts primary color

This gives you:

- a memorable front
- a readable back
- a card that feels consistent with the company profile and design system

## Alternate concept

If you want a cleaner more corporate version:

### Front

- white background
- centered full logo

### Back

- white background
- light gray structure
- small primary accent details

This is safer, but less distinctive than the dark-front version.

## What to prepare before production

- official logo in vector
- official English and Arabic titles
- confirmed phone and email format
- final website URL
- optional QR destination
- printer template if the vendor provides one

## Final export

Export:

- print-ready PDF with bleed
- outlined version if needed by printer
- editable `.ai` master file

Also keep:

- one clean master template
- one version with live editable text
- one packaged final output folder

## Best next step

Create two quick concept directions in Illustrator:

1. dark premium front / light bilingual back
2. white minimal front / white bilingual back

Choose the better direction first, then build the final template and employee variants from that.
