# Verify Final

1. Read the deliverable and every declared input artifact/verification report.
2. Confirm all material facts trace to passed claims.
3. Compare numbers, dates, currencies, names, qualifications, and source links for fidelity.
4. Fail any new unsupported factual claim or hidden upgrade from assumption to fact.
5. Write the independent verification artifact and notify Scribe and Steward. Atlas reads lifecycle state from Steward.
