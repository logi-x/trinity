# Books

Atomic insights extracted from books, one folder per title.

In the **full Cornelius vault** this folder ships pre-seeded with ~140 atomic
notes distilled from decision-science books (Kahneman, Taleb, Mlodinow, Kay &
King, and others). This bundled template ships a **lean seed** — the folder is
here so the structure is in place; populate it by processing your own books.

## How to add a book

1. Create a source note in `../01-Sources/Books/` using
   `_SAMPLE - Book Note Template.md`.
2. Make a folder here named after the book (e.g. `antifragile-taleb/`).
3. Extract one atomic insight per note into that folder — your own words, one
   idea each (see `../02-Permanent/README.md`).
4. Run `/find-connections` to weave the new notes into the graph.
