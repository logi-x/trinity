# Atomic Notes Capture One Idea Each

An atomic note contains a single, self-contained idea expressed in one's own words. Keeping each note focused on one concept makes it easier to link, reuse, and recombine across many contexts.

When a note holds only one idea, it becomes a stable unit of thought that can be referenced from anywhere without dragging unrelated material along with it. Atomicity also forces clarity, because a single idea must be stated plainly enough to stand alone. Over time, a collection of atomic notes behaves like a set of interchangeable building blocks: the same note can support an argument in one project and a contrast in another. This reusability is what distinguishes a knowledge system from a pile of documents, where ideas stay trapped inside the page that first recorded them.

**Why this matters**: Single-idea notes are the smallest reusable unit of knowledge, which is what makes a connected note system possible.
