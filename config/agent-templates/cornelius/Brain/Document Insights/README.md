# Document Insights

Research-finding summaries, one folder per extraction session (dated
`YYYY-MM-DD Topic`). Separate provenance from your own permanent notes: these
are AI-extracted from external papers, books, and articles.

In the **full Cornelius vault** this folder ships pre-seeded with ~57 research
sessions on cognition, decision-making, and behavioral science. This bundled
template ships a **lean seed** — the folder is here so the structure is in
place; populate it with your own research runs.

## How sessions are added

- The `/deep-research` and `/extract-document-insights` skills create a dated
  session folder here and write extracted insights into it.
- Validated insights graduate into `../02-Permanent/` via `/graduate-insights`.
- Sessions are logged in `../05-Meta/Changelogs/`.
