# Logix Virtual Workforce

The simulated leadership team and specialist staff of [[Entities/Organizations/Logix]] — advisory **personas** used to pressure-test decisions through structured disagreement, not real employees. Each file is a stable character with its own bias, vocabulary, and authority boundary; the [[Entities/Agents/Councils/Executive Council|councils]] convene them for board meetings, architecture reviews, and product decisions.
