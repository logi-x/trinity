# Action Tracker

Live view over the [[Actions/Actions.base|Actions]] notes (one note per action). Overdue items
first, then open-by-priority, by-owner, and closed. To add an action, create a note in `Actions/`
with `owner`, `due`, `priority` (critical/high/medium/low), `status` (open/in-progress/closed),
and `source` — it appears here automatically. No more editing a giant table.
