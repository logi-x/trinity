# Prettier

Prettier matters here as the formatting baseline that keeps a large AI-assisted codebase readable and low-friction. It is less about style preference than about reducing unnecessary churn in reviews and conversations.

## Why it matters

- minimizes formatting noise in diffs
- keeps generated or AI-edited code consistent
- reduces time spent debating code shape instead of code behavior
