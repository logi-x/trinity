# Vitest

Vitest is the main test runner discussed in Experts frontend and route-level testing work. It matters because many regressions surface first as contract, validation, or state-flow failures that are easiest to catch in fast test runs.

## Typical uses

- route handler tests
- utility and module tests
- validation logic
- regression coverage after refactors
