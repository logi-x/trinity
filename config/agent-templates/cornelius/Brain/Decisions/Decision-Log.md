# Decision Log

Live view over the [[Decisions/Decisions.base|Decisions]] notes (one note per decision). Decisions
due for review surface first, then all by date. To log a decision, create a note in `Decisions/`
with `decision`, `rationale`, `stakeholders`, `review_by`, and `source`.
