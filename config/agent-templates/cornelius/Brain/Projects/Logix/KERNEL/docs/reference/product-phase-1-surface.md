# Phase 1 Product Surface

After seed + login (`ahmed@logi-x.org` / `logix-dev-2026` in local seed), the internal command center includes:

- Dashboard — live counts, attention lists, recent memories, activity
- Clients — list and Client 360 (contacts, brand, assets, portfolio, projects, proposals, invoices, requests, memory, activity)
- Projects — list and Project 360 (environments, services, commands, health, files, repositories, requests, decisions, runbooks, incidents, agent context, activity)
- Vault — folders, versions, shares, public share links
- Proposals & invoices — workspaces, line items, status actions, payments, timelines
- Requests — kanban, transitions, comments, priority, due dates
- Agents — roster, instructions, skills, tools, access, run history
- Memory — kind-aware search and detail pages with linked records
- Repositories & workflows — registry and designer foundation
- Settings — workspace, team, roles, system info

**Client portal** (`owner@experts.sa` / `experts-dev-2026`): Experts-scoped files, proposals, invoices, requests, and portfolio only.
